# ✅ Django To-Do List Application

A simple **Task Management (To-Do List) web application** built using **Django**.
This application allows users to **create, update, and manage daily tasks** easily through a clean web interface.

---

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* ❌ Delete tasks
* 📋 View all tasks in one place
* ⚡ Built with Django's powerful backend

---

## 🛠️ Technologies Used

* **Python**
* **Django**
* **HTML**
* **CSS**
* **SQLite Database**

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/YalapalliKavya/TODO-List.git
```

---

### 2️⃣ Navigate to the project directory

```
cd TODO-List
```

---

### 3️⃣ Create a virtual environment

```
python -m venv env
```

---

### 4️⃣ Activate the virtual environment

Windows:

```
env\Scripts\activate
```

---

### 5️⃣ Install dependencies

```
pip install django
```

---

### 6️⃣ Run database migrations

```
python manage.py migrate
```

---

### 7️⃣ Start the development server

```
python manage.py runserver
```

Open in browser:

```
http://127.0.0.1:8000
```
